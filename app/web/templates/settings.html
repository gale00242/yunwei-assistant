📱 Telegram 通知配置

配置 Telegram Bot 以接收告警通知。你需要创建一个 Bot 并获取 Bot Token 和 Chat ID。

⏱️ 采集配置

建议值：5-60 分钟

📖 使用说明

Telegram Bot 配置

  1. 在 Telegram 中搜索 @BotFather
  2. 发送 /newbot 创建新 Bot
  3. 按照提示设置 Bot 名称
  4. 复制获得的 Bot Token
  5. 向你的 Bot 发送一条消息
  6. 访问 https://api.telegram.org/bot你的TOKEN/getUpdates 获取 Chat ID

SSH 连接

确保运行运维助手的主机可以 SSH 连接到目标服务器：

  • 密钥认证：将公钥添加到目标服务器的 ~/.ssh/authorized_keys
  • 密码认证：直接在添加服务器时输入密码
